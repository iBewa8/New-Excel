This project presents an interactive HR Analytics Dashboard built to analyze employee data and provide actionable insights on workforce trends, attrition, and demographics.

Key Insights
 • Total Employees and Active Workforce
 • Attrition Rate and Attrition Count
 • Employee Distribution by Gender
 • Attrition Analysis by Department, Job Role, and Age Group
 • Education Level and Marital Status Breakdown
 • Average Employee Age Metrics

Tools & Technologies
 • Microsoft Excel
 • Data Visualization & Dashboard Design
 • HR Data Analytics

Purpose

The dashboard helps HR teams and decision-makers:
 • Identify attrition patterns
 • Understand workforce demographics
 • Improve retention strategies
 • Support data-driven HR planning

Highlights
 • Clean and intuitive dashboard layout
 • Dynamic visualizations for quick insights
 • Business-focused HR metrics
